# Yun's World - 雲的天地

一個現代簡約風格的個人網站 🌿

## 設計理念

- **Ins 風美感**：柔和的漸層背景、充足的留白
- **極簡排版**：乾淨的字體選擇，易讀性優先
- **個人化觸感**：溫暖的色彩（粉紫色調）與微互動
- **響應式設計**：完美支援手機桌面

## 技術栈

- Vanilla HTML + CSS (無外部依賴)
- 單頁設計，載入速度極快

## 使用方式

1. 直接開啟 `index.html` 查看效果
2. 本地服務啟動後訪問 http://localhost:3000
3. 部署到 GitHub Pages / Netlify
